ENGINEERING

Blur

Fall 2016

A simple programming language for fast image manipulation and ASCII art creation. The Blur language provides a set of familiar and intuitive syntactic conventions for handling pixel data, useful in image processing algorithms, image to ASCII conversion schemes, and algorithmic/generative ASCII artworks.

CTIS-Lambda

Aug 2016

A scalable Content-To-Image service using the PhantomJS browser, built for AWS Lambda serverless backend architecture. Service can produce background images or thumbnails from a web content source and automatically drop them into cloud storage. AWS Lambda eliminates scalability issues, making this backend service as easy and pain-free as possible.

Ad Network Radar for Pi-Hole

Jul 2016

A geolocation API for use with the ‘pi-hole’ adblocking DNS software. Allows a user to identify and locate all physical systems serving ads to or collecting browsing data from their local network.

AMuSe Emulator

Jan 2016

AMuSe Emlator, built at Columbia’s Wireless & Mobile Networking Lab, is a web-based visual simulation of the lab’s Adaptive Multicast Services project. The Adaptive Multicast Services project is a WiFi transmission technology that provides a scheme to intelligently process receiver feedback for network adaptation (transmission rate) based on this feedback.

Bellman-Ford

Nov 2015

My socket API implementation of a classic distributed routing algorithm, adapts automatically to handle dynamic network topologies.

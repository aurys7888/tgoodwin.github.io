A site of my own
 This page is an assortment of projects and experiments I've worked on over the past few years. Please explore and enjoy my efforts.

About myself -- I am 20 years old and currently living in New York. My interests primarly involve computers, and are loosely focused on communication, signal processing, and design.

    Visual Media

  • WHAT ARE YOU DOING (Aug 2015) An instant message love story about you and someone else. Built with WebGL.
  • WENT TO JAPAN ( Aug 2015) A photo series final project for CU Japan Study Tour. Also an experiment with JPEG encoding. Built with georg fischer's 'glitch-canvas' library. [Please use Chrome or Firefox for intended results]
  • LIMEKILN (Jan 2015) This happened in Central California below the highway overpass.
  • SCREENPRINT (Spring 2014)With varying degrees of success

    Programming

  • HTTP SERVER (April 2015) Simple, single-threaded web server built from scratch in C.
  • SHORTEST PATH EVER (Fall 2014) Interactive map of the continental US implementing Dijkstra's algorithm in Java.
  • MELODY BOX (Spring 2013) Programmable "music box" GUI in Python (built with/requires Pygame library), generates repeating melodies/sonic textures based on user input.
  • GRAPHICAL HUFFMAN ENCODER (Fall 2014) Visualize the Huffman Coding text compression process in Java.

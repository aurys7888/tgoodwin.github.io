Hi my name is Tim, I am 20 years old and living in New York.
I study electrical engineering and computer science at Columbia, where I also research optical signal processing and photonic devices. I'm interested in communication, design, alternative energy, and future technology. Thank you for the visit and please explore.

    Programming Projects

  • HTTP Server (Spring 2015) A very simple, single-threaded web server built from scratch in C.
  • Shortest Path Ever (Fall 2014) Interactive map of the continental US implementing Dijkstra's algorithm in Java.
  • Melody Box (Spring 2013) Programmable music box GUI in Python (built with/requires Pygame library), generates repeating melodies/sonic textures based on user input.
  • Graphical Huffman Encoder (Fall 2014) A program that visualizes the Huffman coding text compression process in Java.
  • Monte Carlo Pi (Spring 2013) You can approximate the digits of Pi just by throwing darts randomly at a circle inside a square! This Python program throws the darts for you and lets you watch.
